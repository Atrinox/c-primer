# Chapter 1 정리

## 1. Portability
- portability는 프로그램이 다른 시스템에서도 수정 없이 실행될 수 있는 성질
- C는 하드웨어에 가까운 언어지만 표준을 지키면 이식성이 높음

## 2. 파일 종류 차이
- Source file (.c) → 사람이 작성한 코드
- Object file (.o) → 컴파일된 기계어 중간 산출물
- Executable file → 실행 가능한 최종 결과물

## 3. 프로그래밍 7단계
1. 프로그램 정의
2. 설계
3. 코드 작성
4. 컴파일
5. 링크
6. 실행
7. 유지보수

## 4. 컴파일러 역할
- C 코드를 기계어(object file)로 변환

## 5. 링커 역할
- 여러 object file을 합쳐 실행 파일 생성
